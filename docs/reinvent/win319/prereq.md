

**Prerequisites for attendees**

1. Attendees need access to an AWS account with access to provision resources.
2. We will be using the us-east-1 (N. Virginia) region. The following resources will be provisioned during the lab: VPC (1), EC2 instances (3), AWS Managed AD (1), Elastic IP (1).
3. If you are not using a Windows computer, please make sure you have the latest RDP client installed on your computer.
