# Welcome to WIN 306 Migrating Windows workloads to AWS

In this workshop we will focus on two diffent types of migrations strategies using replication. The first lab will focus on re-hosting and the second lab will focus on re-platforming. 

![](/assets/images/WIN306/MigrationStrategies.png)

**[Lab 1 - Re-hosting](https://awsentworkshops.com/reinvent2019/win306/rehostlab)** - In this lab we will use install CloudEndure Agents, and use CloudEndure to migrate source machines to a target AWS Environment. We will also use AWS Systems Manager to install the CloudEndure Agent.
