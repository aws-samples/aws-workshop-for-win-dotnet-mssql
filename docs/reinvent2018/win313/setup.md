# Lab Setup
Before starting the labs, there are a couple of prerequisites and setup steps you need to carry out. If you need a hand, then please ask.

## Prerequisites
1. An [Amazon Web Services (AWS) account](https://aws.amazon.com/account/). We will provide one if you need.
2. An [Azure DevOps](https://azure.microsoft.com/en-gb/solutions/devops/) account.
3. The AWS Tools Extension installed in your Azure DevOps account. Otherwise follow these instructions [these instructions](https://docs.aws.amazon.com/vsts/latest/userguide/getting-started.html#install-the-aws-tools-for-vsts-extension) to install the extension.
