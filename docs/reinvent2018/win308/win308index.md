# WIN308 - Hands-On: Automating AWS Infrastructure with PowerShell

## Welcome!
In this Workshop, we will introduce you to the AWS Tools for Windows PowerShell/PowerShell Core and discuss the fundamentals for automating infrastructure on AWS using PowerShell. We have provided a set of challenges that you can choose from as a way to help you learn how you can use PowerShell to automate infrastructure on AWS in a variety of real-world scenarios.
